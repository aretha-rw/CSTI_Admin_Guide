Common terms

All definitions refer to the term’s technical use and are specific to the CSTI.

Class: A specific, scheduled instance of a course- class is to course as “Denver” is to “city.”

Coach: A learner that supervises other learners and has access to the profiles of each learner they supervise.

Course: The template from which specific classes are built.

Hybrid: An object which groups two or more courses together into a single unit. When a learner registers for a hybrid class, they automatically register for all its components at once. A hybrid class is considered incomplete unless all its component classes are completed.

LMS: Abbreviation of Learning Management System. Most segments of the website that you will work with are part of the LMS, including the profile and all consoles.

Module: Refers both to the piece of a hybrid object that links its components and to a section of a web-based training which contains multiple SCORM packages.

Role: Roles are sometimes also referred to as permissions; a learner’s roles grant them access to consoles and their functions.

SCORM: Abbreviation of Sharable Content Object Model. A standard format for interactive online trainings.

Template: The object on which individual hybrid classes are based. Template is to Hybrid as Course is to Class.

Training Organization: An organization which develops and facilitates classes on behalf of the CSTI and serves as a contact for training-specific questions. The contact designated for each training organization will appear in all registration-related communications.

WBT: Abbreviation of Web-Based Training. Web-based trainings are completed by accessing a SCORM package, which then reports back completion data to the LMS. Webinars and video-conferences are not WBTs.
