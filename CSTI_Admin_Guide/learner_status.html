Appendix: Learner Statuses

Non-Final Statuses

Learners in these statuses will see their class in the Upcoming Classes section of their profile. If a class has one or more learners in one of these statuses, it cannot be closed.

Registered: The learner is currently enrolled and awaiting a class. For WBTs, learner has immediate access to their training.

Waiting: The learner has attempted to register for a class that was full and will be moved to Registered if an opening appears.

Pending: The learner has attempted to register for a class that has special registration requirements and will be moved to Registered if those requirements are fulfilled.

Final Statuses

Learners in these statuses will see their class in the History section of their profile.

Completed: The learner has finished a class and met all requirements for receiving credit. This is the only status that bestows training hours.

Cancelled: The learner was in a non-final status for a class that was cancelled.

Dropped: The learner has either dropped or been dropped from a class. Conventionally, drops are entered prior to a class to allow waiting learners to be registered. The supervisor of a learner who is dropped from a class will be notified. If the class has already been closed, one of the following statuses is more appropriate.

No Show: The learner did not attend a class and did not submit a drop. A notification is automatically sent to the supervisor of any learner marked No Show.

Waived: The learner did not attend a class and did not submit a drop, but an appropriate explanation was given and No Show status is not appropriate.

Incomplete: A learner attended or began a class but left some component unfinished. For example, a learner fully attends the classroom portion of a Hybrid but not the WBT, or attends a classroom-only training but leaves several hours early.
